📊 Análise Exploratória de Dados — Titanic Dataset

Projeto de análise exploratória (EDA) utilizando Python, pandas e matplotlib sobre o clássico dataset do Titanic.

🎯 Objetivo

Demonstrar habilidades em análise de dados com Python: exploração de base, tratamento de dados nulos, agrupamentos e visualização de insights.

🔍 O que foi analisado


Estrutura e qualidade da base (shape, isnull, describe)
Taxa de sobrevivência geral e por classe socioeconômica
Diferença de sobrevivência por gênero e faixa etária
Visualizações com matplotlib para comunicar os padrões encontrados
